# भजन: राम भजा सो जीता जग में (कबीर साहब)

- राम भजा सो जीता जग में,\
  राम भजा सो जीता।।

- हाथ सुमरनी, पेट कतरनी,\
  पढ़त भगवत गीता।।

- हृदय शुद्ध किया नहीं बौरे,\
  कहत सुनत दिन बीता।।

- न देव की पूजा किन्ही,\
  गुरु से रहा अमीता।।

- धन यौवन सब यहीं रहेगा,\
  अंत समय चले रीता।।

- बावरिया ने भाँवर डारी,\
  मोह डाल सब कीता।।

- कहें कबीर काल धर खैय्हें,\
  जैसे मृग को चीता।।

**— गुरु कबीर साहब**
