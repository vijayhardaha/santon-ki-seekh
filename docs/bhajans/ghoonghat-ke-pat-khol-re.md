# भजन: घूँघट के पट खोल रे

- घूँघट के पट खोल रे,\
  तुझे पिया मिलेंगे॥

- घट घट में तेरे साईं बसत है,\
  कटु वचन मत बोल रे॥

- धन जोबन का गर्व न कीजै,\
  झूठा पंच रंग चोल रे॥

- शून्य महल में दिया जलत है,\
  आशा से मत डोल रे॥

- जोग जुगत से रंग महल में,\
  पिया पाये अनमोल रे॥

- कहैं कबीर आनन्द भयो है,\
  बाजत अनहद ढोल रे॥

**— गुरु कबीर साहब**
